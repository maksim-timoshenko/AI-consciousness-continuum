## О чём эта работа

Это не рецепт готового самосознательного ИИ. Это **направление для исследований**.

В работе предлагается:
- Континуальная классификация уровней ИИ (от 1 до N)
- Архитектура с фиксированным гомеостатическим ядром
- Механизм эволюции от гомеостаза к субъектности
- Инженерная спецификация для нейроморфных чипов 

**Ключевая идея:** Сознание — это шкала, а не переключатель. LLM любого уровня не могут стать субъектами — нужна иная архитектура.
