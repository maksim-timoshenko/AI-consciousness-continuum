## What This Work Is About

This is not a recipe for a ready-made self-aware AI. It is a **direction for research**.

The work proposes:
- A continuum classification of AI levels (from 1 to N)
- An architecture with a fixed homeostatic core
- A mechanism for evolution from homeostasis to subjectivity
- An engineering specification for neuromorphic chips and diffusion memristors

**Key Idea:** Consciousness is a spectrum, not a threshold. LLMs of any scale cannot become subjects — a different architecture is required.
